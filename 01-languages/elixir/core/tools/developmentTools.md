## ElixirLS

Para garantizar una experiencia de codificación fluida y profesional en Elixir, es altamente recomendable instalar el plugin ElixirLS (Elixir Language Server). Este servidor de lenguaje proporciona funcionalidades cruciales que transforman cualquier editor de texto en un IDE robusto, tales como:

- Autocompletado inteligente de código.

- Inspección y navegación de módulos.

- Diagnóstico de errores en tiempo real.

- Funcionalidades de debugging (depuración).

### Compatibilidad

La gran ventaja de ElixirLS es su amplia disponibilidad. Lo puedes encontrar e instalar directamente en:

- Visual Studio Code (VS Code), el editor más popular.

- IDEs derivados o compatibles con el ecosistema de VS Code, como Cursor, VSCodium, Antigravity, entre otros.
